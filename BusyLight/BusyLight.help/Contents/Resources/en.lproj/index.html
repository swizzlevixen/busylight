Busy Light Help

Busy Light is a macOS menu bar app that connects to Home Assistant to activate scenes, primarily for controlling a "busy light" outside a room to indicate whether you may be disturbed.

Select a topic below to learn more:

  • Getting Started — connect to Home Assistant and set up the app
  • Adding Scenes — choose which scenes appear in the menu bar
  • Using Triggers — automatically activate scenes based on system events
  • Troubleshooting — solutions for common issues
