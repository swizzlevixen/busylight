Busy Light Help

Using Triggers

Triggers let Busy Light automatically activate a scene when something happens on your Mac, such as turning on the webcam for a video call. This is the key feature that makes your busy light work hands-free.

Available triggers

Busy Light monitors four types of system events:

  • Webcam — fires when any camera on your Mac starts or stops being used by an app.
  • Microphone — fires when any microphone starts or stops being used.
  • Screen Lock — fires when you lock the screen or when it unlocks.
  • Focus Mode — fires when a system Focus (e.g. Do Not Disturb) turns on or off. This trigger is experimental and may not work on all macOS versions.

Set up a trigger

  1. Open Settings… and select the Triggers tab.
  2. Find the trigger you want (e.g. Webcam → On) and turn on its toggle.
  3. Choose a scene from the dropdown. That scene will be activated whenever the trigger fires.

On and Off triggers

Each trigger type has two states: On (e.g. camera starts) and Off (e.g. camera stops). You can configure each independently:

  • Set the On trigger to activate your "busy" scene when a call starts.
  • Set the Off trigger to activate your "available" scene when the call ends.

Trigger priority

When multiple triggers fire at the same time, the webcam trigger takes priority over the microphone trigger. For example, if both the camera and microphone turn on simultaneously (as in a video call), the webcam scene is used.

Triggers work in the background regardless of which app is in the foreground. The webcam and microphone monitors respond to device status changes in real time using system event callbacks.
