Busy Light Help

Adding Scenes

Scenes in Home Assistant capture the state of one or more devices (lights, switches, etc.) so they can be restored with a single action. Busy Light lets you pick which scenes appear in the menu bar for quick activation.

Open the Scenes tab

Click the Busy Light icon in the menu bar, choose Settings…, then select the Scenes tab.

When the tab opens it automatically fetches the latest scenes from your Home Assistant instance. You can also click Refresh Scenes from Home Assistant at any time to update the list.

Add a scene

  1. Click the + button below the scene list.
  2. Select a scene from the pop-up menu. Scenes already in the list are dimmed.
  3. The scene appears at the bottom of the list, ready to customize.

Add a divider

Dividers help organize the menu when you have many scenes. Click the + button and choose Add Divider. The divider appears as a horizontal line in both the list and the menu bar dropdown.

Customize a scene

  • Emoji — click the emoji button to open the system Character Viewer and pick any emoji.
  • Display Name — type a friendly name that appears in the menu (independent of the Home Assistant entity name).

Reorder scenes

Drag a scene up or down in the list to change the order it appears in the menu bar dropdown.

Remove a scene

Select a scene in the list and click the − button to remove it. This only removes it from the menu — the scene still exists in Home Assistant.

Keyboard shortcuts

You can assign a global keyboard shortcut to any scene so it can be activated from anywhere, even when the menu is closed.

  1. Click the shortcut area next to a scene (it shows None if no shortcut is set).
  2. Press the key combination you want to use (e.g. ⌘1).
  3. The shortcut is registered immediately and also appears next to the scene in the menu bar dropdown.

Press the same shortcut again while the scene is active to deactivate it.

Display mode

In the General tab of Settings you can choose how scenes are displayed in the menu bar:

  • Emoji Only — shows just the emoji.
  • Name Only — shows the display name.
  • Both — shows the emoji followed by the name.
